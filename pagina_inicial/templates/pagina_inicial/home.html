Sobre o Projeto

Este trabalho acadêmico, sob a disciplina do Projeto Integrador do curso de Big Data para Negócios da FATEC Ipiranga, tem como finalidade principal a análise de um caso da empresa EGA Soluções Industriais, que produz softwares de controle de chão de fábrica (MES). Nesse projeto, o grupo lançou mão de ferramentas de Big Data, incluindo Cálculos Estatísticos, Inteligência Artificial, Modelagem de Dados, Dashboards e criação de um website.

Sobre a Empresa

A EGA Soluções Industriais é uma empresa brasileira fundada em 1988, que atua há mais de três décadas no desenvolvimento de soluções tecnológicas voltadas para a gestão da produção industrial. Com sede no Brasil e capital 100% nacional, a empresa se consolidou como uma fornecedora de sistemas MES (Manufacturing Execution System), ferramentas fundamentais para o monitoramento e controle em tempo real das operações de manufatura.

Seu foco é oferecer soluções completas que integram hardware e software, possibilitando às indústrias maior eficiência operacional, redução de desperdícios e aumento da produtividade, alinhando-se às demandas da Indústria 4.0.

O portfólio da EGA abrange funcionalidades que permitem acompanhar o desempenho das máquinas e processos, calcular indicadores como OEE, gerenciar qualidade, rastrear produtos, controlar inventários e monitorar o consumo de energia. Suas soluções integram-se com sistemas ERP e incluem recursos móveis para acompanhamento remoto.

Sobre o Site

O website foi desenvolvido utilizando Python e Django. Nele, é possível realizar cadastro e login, caso o usuário já possua uma conta. No cadastro, é necessário informar nome de usuário, nome, sobrenome, e-mail, senha e confirmação de senha. O sistema valida cada campo, exibindo mensagens específicas quando algo precisa ser corrigido.

No login, caso o usuário informe credenciais incorretas, o sistema também exibe mensagens orientativas. Há ainda a opção de logout, permitindo que o usuário encerre a sessão.

O site permite acessar as análises de Machine Learning desenvolvidas no projeto, conectadas diretamente ao banco de dados, de forma que qualquer alteração impacta automaticamente as análises. Também é possível visualizar os Dashboards integrados do Power BI, atualizados dinamicamente.

Para acessar as análises ou dashboards, é necessário estar logado. Há ainda uma página administrativa restrita, onde é possível visualizar o banco de dados e os usuários cadastrados. O website está hospedado na Vercel, permitindo acesso público pelo link disponibilizado.
